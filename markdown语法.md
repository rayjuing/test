# 列表

## 退出列表的方法

1.两次回车

2.ctrl + [  退出有序列表

3.ctrl + ] 退出无序列表

## 有序列表 (ctrl + shift +[)

使用 ctrl + [ 来结束列表 或者两次回车

把大象放进冰箱:

1.  把冰箱打开
2. 把大象放进去
3. 关上冰箱

## 无序列表 (ctrl + shift + ])

或者使用 -加空格 来实现

无序列表

- 撒旦 
- 放到

- 深度
